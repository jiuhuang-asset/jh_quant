# 资产定价模型因子配置说明文档

## 1. Fama-French 三因子模型 (FF3)
**学术来源**: Fama and French (1993)
**核心思想**: 认为股票收益率除了受市场风险影响外，还受公司市值规模和账面市值比的影响。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **Mkt** | 市场因子 | 市场组合收益率减去无风险利率 ($R_m - R_f$)。反映系统性风险。 |
| **SMB** | 规模因子 | **Small Minus Big**。做多小市值组合，做空大市值组合。捕捉小盘股长期跑赢大盘的溢价。 |
| **HML** | 价值因子 | **High Minus Low**。做多高 B/M (账面市值比) 股票，做空低 B/M 股票。捕捉价值股相对于成长股的溢价。 |

---

## 2. Fama-French 五因子模型 (FF5)
**学术来源**: Fama and French (2015)
**核心思想**: 在三因子的基础上增加了盈利和投资两个维度。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **RMW** | 盈利因子 | **Robust Minus Weak**。做多盈利能力强（高营业利润率）的公司，做空弱者。逻辑是高盈利公司未来预期回报更高。 |
| **CMA** | 投资因子 | **Conservative Minus Aggressive**。做多投资稳健（低资产增长率）的公司，做空投资激进的公司。逻辑是过度扩张通常降低股东回报。 |

---

## 3. Carhart 四因子模型 (CARHART)
**学术来源**: Carhart (1997)
**核心思想**: 认为动量效应是除规模、价值之外最重要的异象。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **UMD** | 动量因子 | **Up Minus Down**。做多过去一年涨幅前 30% 的股票（不含最近一个月），做空跌幅前 30% 的股票。捕捉“强者恒强”的趋势效应。 |

---

## 4. Novy-Marx 四因子模型 (NOVY_MARX)
**学术来源**: Novy-Marx (2013)
**核心思想**: 重新定义了盈利因子，认为毛利（Gross Profit）比净利更难被操纵且具有更强的预测力。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **GP/A** | 毛利因子 | **Gross Profit / Total Assets**。该模型的核心。Novy-Marx 认为这是最“纯净”的盈利指标，应结合行业中性化的 HML 使用。 |
| **HML_Adj** | 行业调整价值 | 价值因子在行业内进行标准化，避免行业偏好造成的价值陷阱。 |

---

## 5. q-factor 四因子模型 (HOU_XUE_ZHANG)
**学术来源**: Hou, Xue, and Zhang (2015)
**核心思想**: 基于投资理论（q-theory）而非经验观察。认为投资和 ROE 直接决定了公司的贴现率。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **ME** | 市值因子 | 类似 SMB，但分组逻辑（2x3x3）更细致。 |
| **IA** | 投资增加额 | **Investment-to-Assets**。计算总资产的年增长率。低投资的公司预期收益更高（均衡状态下）。 |
| **ROE** | 盈利因子 | 使用最近一期的**单季 ROE**。该模型认为盈利的变动能迅速反映在股价中。 |

---

## 6. Daniel-Hirshleifer-Sun 行为模型 (DHS)
**学术来源**: Daniel et al. (2020)
**核心思想**: 基于投资者心理（过度自信和有限注意力）构建模型，弃用了传统的规模和价值。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **PEAD** | 盈余漂移 | **Post-Earnings Announcement Drift**。利用 SUE（标准意外盈余）构建，捕捉市场对财报消息反应不足的红利。 |
| **FIN** | 融资因子 | **Financing Factor**。基于公司在公开市场的融资行为（增发、回购）。融资越多的公司，未来表现往往越差。 |



## 7. 中国三因子模型 (CH-3)
**学术来源**: Liu, Stambaugh, and Yuan (2019) / 汪昌云等
**核心思想**: 针对 A 股高度散户化及“壳价值”异象，通过过滤小市值干扰来重构价值因子。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **SMB** | 规模因子 | 与 FF3 类似，但通常在剔除壳公司后计算，更能代表真正的规模效应。 |
| **VMG** | 价值因子 | **Value Minus Growth**。不同于 HML，它在分组前先剔除市场市值最小的 30% 股票（壳资源密集区）。捕捉剔除投机因素后的真实价值溢价。 |

---

## 8. Stambaugh-Yuan 四因子模型 (SY4)
**学术来源**: Stambaugh and Yuan (2017)
**核心思想**: 将数十个市场异象聚类为两个“误定价”因子，反映管理偏差和经营绩效偏离。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **MGMT** | 管理因子 | **Management Cluster**。综合了资产增长率、应计项、净股票发行等指标。反映管理层过度扩张或操纵利润导致的误定价。 |
| **PERF** | 绩效因子 | **Performance Cluster**。综合了 ROE、资产周转率变动、财务困境指标。捕捉基本面强劲但被市场忽视的溢价。 |

---

## 9. 短期反转模型 (REVERSAL)
**学术来源**: 经典市场微观结构研究（在 A 股表现极强）
**核心思想**: 捕捉散户过度交易导致的短期价格超摆。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **REV** | 反转因子 | **Short-term Reversal**。通常计算过去 20 个交易日（一个月）的累计收益率。做多跌幅最大的组，做空涨幅最大的组。 |

---

## 10. 低波动模型 (LOW_VOL)
**学术来源**: Ang et al. (2006) / 波动率悖论
**核心思想**: 挑战“高风险高收益”，发现低波动股票在 A 股具有更优的风险调整后收益。

| 因子名称 | 因子含义 | 作用与计算逻辑 |
| :--- | :--- | :--- |
| **IVOL** | 特质波动率 | **Idiosyncratic Volatility**。通过 CAPM 或 FF3 模型对日收益率做残差回归，取残差的标准差。捕捉无法被系统风险解释的异常波动稳定性。 |

## 因子计算实施逻辑 (Implementation Rules)

1.  **分组逻辑 (Portfolio Sorting)**:
    * **2x3 分组**: 先按市值（Size）中位数分为 S/B 两组；再按目标因子（如 BM）的 30%/70% 分位数分为 L/M/H 三组。因子的收益率为 $(S/H + B/H)/2 - (S/L + B/L)/2$。
2.  **权重计算 (Weighting)**:
    * **Value-Weighted**: 组合内个股收益率按其流通市值的占比进行加权求和。这是学术界抵御“小票效应”干扰的标准做法。
3.  **数据频率**:
    * 市值数据：每日/每月更新。
    * 财报数据（ROE, GP/A, IA）：按季报公告日（A 股需处理 4/30, 8/31, 10/31 等截止日）触发更新。
